# Asteroid Game

A simple **Asteroids-style game** built with **Python** and **Pygame** as part of the **Boot.dev curriculum**.

## Features
- Player-controlled spaceship
- Asteroid movement and splitting
- Shooting mechanics
- Collision detection

## Requirements
- Python 3
- Pygame

   cd asteroid-game
